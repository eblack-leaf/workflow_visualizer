Crate workflow_visualizer

source ·
Expand description

Workflow Visualizer is a lib for cross-platform declarative UI It is comprised of three major parts. trait Workflow to setup application messaging. Visualizer which handles rendering Runner which invokes the visualizer’s tools for multiple platforms
